## 学习目标   
- 主要是为了修改界面上的图标，但是图标的制作，经常用到2款软件：    
	- adobe  photoshop：简称PS，主要用于处理位图  
	- adobe illustrator：简称AI，主要用于处理矢量图   
- 一般是两款软件配合使用，在PS中画图，在AI中进行矢量化处理
